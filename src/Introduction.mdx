# Nimbus Design System

The single source of truth for the **Nimbus** brand — design tokens, foundations,
reusable UI components, application patterns, and ready-to-use product screens, all
themed from the Nimbus Figma token export.

Built with **React 19 · TypeScript · Tailwind CSS v4 · React Aria · Recharts ·
Storybook 10 · Vite**.

## Brand & tokens

The theme comes from the Nimbus design tokens in `reference/styles/` and is applied in
`src/styles/theme.css`:

- **Brand** — pink primary (`#DA6EA3` / `--color-brand-500`) with teal (`#37B6B7`) as the secondary / `_alt` accent.
- **Neutrals** — a warm gray scale for surfaces, text, and borders.
- **Typography** — Proxima Nova on the Nimbus type scale (`text-md` 15px, display sizes up to 72px).
- **Radius & spacing** — Nimbus token values (e.g. `radius-md` 8px).

Reuse the theme classes (`bg-brand-solid`, `text-primary`, `border-secondary`,
`text-md`, `radius-md`, …) and new work inherits the Nimbus brand for free.

## Getting started

```bash
npm install              # install dependencies
npm run storybook        # launch this library at http://localhost:6006
npm run build-storybook  # static build → ./storybook-static
```

Add a component or screen with a matching `*.stories.tsx` and it appears in the correct
category automatically. Every push to `main` redeploys the live Storybook to GitHub Pages:

**→ https://girardjustin1.github.io/nimbus-ds-v1/**
